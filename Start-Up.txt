Start-Up 

Prerequisites
- Node.js 
- npm 

Steps to Run the Server

- Create a .env File in the server folder copy this into it: 
MONGODB_URI="mongodb+srv://samuel:lqBvVKDKsE6MbvOZ@backend.psehq.mongodb.net/"
PORT=8080
DEFAULT_BRANCH = "BranchSheffield"
- cd server
- npm i 
- npm run dev

- ctrl c in terminal to stop server


Steps to Run the React Application

- cd libary-management-system
- npm i 
- npm start

- ctrl c in terminal to stop react app

Additional Notes

- Ensure that both the server and the React application are not configured to run on the same port.
- Ensure all dependencies are correctly installed.
